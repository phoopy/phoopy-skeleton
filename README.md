# Phoopy Skeleton

## Install

```bash
curl https://raw.githubusercontent.com/phoopy/phoopy-installer/master/phoopy-installer -o phoopy-installer
./phoopy-installer 1.1.0 my-project
```

## Run

```bash
bin/console
```

## Run tests

```bash
pip install -r requeriments-development.txt
phulpy test
```
